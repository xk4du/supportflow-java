# 🎫 SupportFlow - Java Ticket System

SupportFlow is a simple console-based ticket management system developed in Java.
The project was created to practice fundamental programming concepts such as object-oriented programming, lists, methods, user input, and basic system flow.

---

## 📌 About the Project

This application simulates a basic support system where users can create tickets and the support team can manage and respond to them.

The project runs entirely in the terminal and does not use any database or external framework, making it ideal for Java beginners.

---

## 🚀 Features

* 👤 User area
* 🛠️ Support area
* 📝 Create support tickets
* 📋 List user tickets
* 📂 List all tickets
* 💬 Reply to tickets
* 🔄 Update ticket status
* ✅ Close tickets
* 🧭 Simple console menu
* 🧱 Object-oriented structure

## ▶️ How to Run

Make sure you have Java installed on your machine.

To check your Java version, run:

```bash
java -version
```

Compile the project:

```bash
javac *.java
```

Run the project:

```bash
java Main
```
